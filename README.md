# 📊 Controle de Presença - ExpliCAASO

Sistema de controle de presença com PostgreSQL e backup no Google Drive.

## Instalação

```bash
# Clone o repositório
git clone https://github.com/SEU-USUARIO/controle-presenca.git
cd controle-presenca

# Execute o instalador
./scripts/setup.sh

# Inicie o sistema
source venv/bin/activate
python src/controle_presenca/main.py
